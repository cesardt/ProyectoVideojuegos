-----------------------------Multiusuario--------------------------------------
Para hacer que funcione la parte de multiusuario debes:

Debes crear un cuarto que se llame TEC y una zona: ITESM....

Deben tener xamp en su computadora con root nada mas como usuario nada en el pass (es la configuración inicial)

Ya que este eso, abren xamp habilitan el servidor y mysql... y se meten a localhost/phpmyadmin

En esa sección importa el archivo adjunto "videojuego.sql"

Se incluye un zis que tiene el jar que debes montar en tu servidor para manejar las solicitudes del juego.

------------------------------Moviemientos Basicos---------------------------

Brincar.- Con la barra espaciadora
Disparar.-Tecla "Z"
Caminar.- Con las Flechas (izq para ir atrás, o der para ir adelante)
Doble Salto.- Presiona dos veces barra espaciadora.
